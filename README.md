# Avaliação de Qualidade: Mural UnB

Essa avaliação faz parte da disciplina Qualidade de Software, onde se realiza uma avaliação de qualidade do software **Mural UnB**. O software consiste em uma centralização digital de oportunidades acadêmicas e extracurriculares da Universidade de Brasília, que utiliza inteligência artificial e uma interface intuitiva para conectar estudantes a projetos, laboratórios e eventos.

O projeto é mantido por uma equipe multidisciplinar de estudantes da FGA, com o foco de criar uma solução aberta e colaborativa. Toda a documentação e artefatos de avaliação são construídos e disponibilizados por meio deste repositório.

## Como rodar o projeto

Este projeto utiliza o [Docusaurus 3](https://docusaurus.io/), um gerador de sites estáticos moderno. Para rodar a documentação localmente, siga os passos abaixo:

### Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 18 ou superior recomendada)
- `npm` ou `yarn` instalados.

### Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/FCTE-Qualidade-de-Software-1/2026-1_T01_GRACE_HOPPER.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd 2026-1_T01_GRACE_HOPPER
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```

### Iniciando o servidor de desenvolvimento

Para rodar o site localmente e ver as alterações em tempo real:

```bash
npm start
```

_Isso iniciará um servidor de desenvolvimento local e abrirá uma aba no seu navegador padrão (geralmente em `http://localhost:3000`). O site será recarregado automaticamente sempre que você salvar alterações nos arquivos._
