# 2. Descrição Estruturada do Software

O Mural UnB é uma plataforma web desenvolvida para a comunidade da Universidade de Brasília, utilizada para divulgação e compartilhamento de informações acadêmicas e sociais. O sistema permite que usuários publiquem anúncios, divulguem eventos, ofereçam serviços, compartilhem oportunidades e realizem buscas por conteúdos de interesse dentro do ambiente universitário.

A aplicação é acessada diretamente pelo navegador, em dispositivos desktop e móveis, caracterizando-se como um sistema web colaborativo. Entre suas funcionalidades principais estão autenticação de usuários, criação e gerenciamento de publicações, pesquisa por categorias e navegação responsiva. Sua estrutura funcional é composta, principalmente, pelos módulos de usuários, publicações, busca e interface web, integrados a um sistema de armazenamento de dados e serviços de hospedagem online.
